# 💰 FinCalc

Aplicação web financeira pessoal com duas funcionalidades principais: calculadora e conversor de divisas com taxas de câmbio em tempo real.

## 🚀 Demo
[Ver aplicação ao vivo](https://fincalc-rho.vercel.app)

## ✨ Funcionalidades
- 🧮 Calculadora completa com suporte a teclado
- 💱 Conversor de divisas com taxas em tempo real
- 🌙 Modo escuro / claro
- 📱 Design responsivo para mobile e desktop

## 🛠️ Tecnologias
- React 19
- Vite
- Axios
- API REST externa (open.er-api.com)
- CSS puro com variáveis e animações

## ▶️ Como correr localmente
```bash
npm install
npm run dev
```
